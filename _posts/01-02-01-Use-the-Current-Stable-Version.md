## Use the Current Stable Version (5.4)

If you are just getting started with PHP make sure to start with the current stable release of [PHP 5.4][php-release]. PHP has made great strides adding powerful [new features](#language_highlights) over the last few years. Don't let the minor version number difference between 5.2 and 5.4 fool you, it represents _major_ improvements. If you are looking for a function or it's usage, the documentation the [php.net][php-docs] website will have the answer.

[php-release]: http://www.php.net/downloads.php
[php-docs]: http://www.php.net/manual/en/
